Meowmood

Meowmood is an interactive AI application that detects user facial expressions and hand gestures in real time and responds with corresponding cat reactions. It combines computer vision and AI to create a playful and engaging human-computer interaction experience.

The system interprets emotions and gestures to generate dynamic visual feedback, making digital interactions more expressive and entertaining.

Features
- Real-time face detection  
- Hand gesture recognition  
- Emotion-based cat reactions  
- Instant visual feedback  

Uses
- Interactive entertainment applications  
- Creative AI-based user experiences  
- Gesture and emotion recognition demos  
- Educational projects in computer vision
